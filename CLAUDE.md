# 絶対ルール: 個人情報の取り扱い（最優先・例外なし）

このリポジトリは**公開リポジトリ**である。以下のルールを必ず厳守すること。

1. **あらゆる個人情報を GitHub に絶対にあげない。**
   対象はすべての書き込み先: ファイル内容・コミットメッセージ・コミットの author/committer・ブランチ名・Issue / PR のタイトルと本文・コメント。
   個人情報の例: 氏名、個人や勤務先のメールアドレス、電話番号、住所、生年月日、パスポート番号、予約番号、搭乗者名、勤務先が特定できる情報。
2. コミットに使ってよいメールアドレスは次の匿名アドレスのみ:
   - `28840183+ichi-kato@users.noreply.github.com`
   - `noreply@anthropic.com` / `noreply@github.com`
3. push する前に、送信されるコミットの diff・メッセージ・author/committer に個人情報が含まれていないか必ず確認する。
4. **迷ったら上げない。** 個人情報かどうか判断がつかない場合は、必ずユーザーに確認してから進める。

このルールは `.claude/hooks/check-pii.sh`（PreToolUse フック）でも機械的に強制される。
フックがブロックした場合は、回避策を探さず、原因となった個人情報を取り除くこと。
